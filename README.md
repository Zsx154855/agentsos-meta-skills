# AgentsOS Meta Skills

> AI 产品建造操作系统：四个 Claude Code Skills 覆盖完整建造链路。

## 技能矩阵

```
Karpathy 编程纪律      →  如何正确地建造
tonny Builder 视角     →  建造什么、怎么变现
AgentsOS 安全 Gate     →  建造完了怎么安全交付
Skill 工作台           →  把以上串成自动化管道
```

## 安装

```bash
npx skills add Zsx154855/agentsos-meta-skills
```

## 四个 Skill

| Skill | 触发词 | 用途 |
|-------|--------|------|
| **karpathy-coding-discipline** | 「用 Karpathy 的编程纪律」「简化这个实现」 | 审查代码、从零实现、最小化抽象 |
| **tonny-claude-ai-builder** | 「用 tonny 的视角」「AI 工具怎么赚钱」 | 找到可建造的痛点、变现路径 |
| **agentsos-security-gate** | 「执行安全 Gate」「安全硬化」 | 密钥治理、权限最小化、工作区清理 |
| **agentsos-skill-workbench** | 「skill 工作台蒸馏 XX」 | 女娲蒸馏 → 安全验证 → 商业化评估 |

## 建造链路

```
输入（人名/主题/痛点）
  → 工作台启动管道
    → 女娲蒸馏（提取心智模型）
    → 安全 Gate（扫描+验证）
    → tonny 评估（商业化和分发判断）
  → 产出：可安全分发的 AI Skill
```

## 来源

由 AgentsOS 项目通过 6 个安全 Gate 的实战验证提炼。详见 `skills/agentsos-security-gate/SKILL.md` 中的完整溯源。
