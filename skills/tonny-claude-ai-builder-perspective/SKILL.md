**局限**：纯杠杆思维可能低估需要深度思考而非建造的任务价值；不是所有知识工作都适合转化为自动化。

## 五步建造管道（5-Step Build Pipeline）

课程核心方法论——4 小时 10 分钟从零到可盈利 AI 工具：

### Step 1: 环境搭建 + 上下文工程 (~30min)
- 安装 Claude Code CLI，配置 VS Code
- 创建 CLAUDE.md（项目大脑）——定义上下文和 AI 行为
- 配置 Git、API Key、基础 MCP 连接

### Step 2: Vibecoding 快速原型 (~60min)
- 用自然语言描述工具需求
- 让 Claude 生成完整实现
- 截图迭代循环：给 Claude 看截屏 → 让它调整
- **核心原则**：不手写代码，指挥 AI 写

### Step 3: 自动化系统设计 (~60min)
- 将单次工具升级为自动化工作流
- 通过 MCP 连接外部数据源
- 设置定时执行（headless 模式 / cron / n8n）
- 添加错误处理和日志

### Step 4: 收入接入 (~45min)
三种变现模式：
- **直接接单**：把工具包装成服务 → $50-150/hr
- **SaaS / 产品化**：Stripe + Supabase 持续付费 → $500-5000+
- **模板 / 数字商品**：出售 skill 或工作流 → $24-109

### Step 5: 部署与分发 (~45min)
- 部署到 Vercel / GitHub Pages / Headless
- 创建基础落地页
- 设置使用追踪和计费
- 部署 → 获客 → 迭代反馈循环
